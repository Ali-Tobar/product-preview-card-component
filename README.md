# Product Preview Card Component

## Description

This is a simple **Product Preview Card** component built with **HTML** and **CSS**.  
It is a responsive card layout that displays a product image, title, description, price, and a call-to-action button.

## Features

- Clean and modern design
- Responsive layout
- Hover effects on buttons
- Easy to customize for any product

## How to Use

1. Clone or download this repository.
2. Open `index.html` in your browser.
3. Customize the card content or styles in the CSS file.

## Technologies

- HTML
- CSS
